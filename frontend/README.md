# WhatATrade! — Frontend

React 18 web app for the WhatATrade! trading journal.

## Quick start
```bash
npm install
npm run dev     # → http://localhost:3000
npm run build   # production build → /dist
```

## Key files
- `src/components/ui/Logo.jsx` — WhatATrade! logo (concept C)
- `src/store/useStore.js`      — Zustand global state
- `src/services/api.js`        — all backend API calls
- `src/pages/`                 — one file per route

See root `README.md` for full documentation.
