# WhatATrade! — Backend

Node.js + Express API for the WhatATrade! trading journal.

## Quick start
```bash
cp .env.example .env    # fill in broker keys when ready
npm install
npm run dev             # → http://localhost:3001
```

## Health check
```bash
curl http://localhost:3001/health
```

See root `README.md` for full API documentation and broker setup.
